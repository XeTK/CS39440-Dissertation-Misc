• get /login

        • Logs the user into the application

      • post /login

        • Login Operation

      • post /register

        • This is registering the user

      • get /logout

        • This destroys the users session

      • get /user

        • Returns the user object for the current user

      • get /user/id/{user}

        • Return user for given user_id

      • get /user/username/{user}

        • Returns user object by users username

      • get /friends

        • Returns a list of the users friends

      • post /friend/add

        • This is adding a new friend

      • post /friend/remove

        • This is removing a new friend

      • get /comments/{message_id}

        • Get the votes for messages or comments

      • post /comments/add

        • This is adding a new comment

      • get /message/{id}

        • Get message by id

      • get /message/friends

        • Get messages for current user left by friends

      • get /messages/notifications

        • Return a list of the messages that haven not been read

      • get /messages/user

        • Return a list of the messages for the current user

      • post /message/read

        • Marking a message as read

      • post /message/add/message

        • Adding a new message

      • post /message/add/user

        • Adding visibility to a message to a user

      • post /vote/{type}/add

        • This adds a vote to a message or a comment

      • get /vote/{type}/{id}

        • Return the statistics for either a comment or message from votes
